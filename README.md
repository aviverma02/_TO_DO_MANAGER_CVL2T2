# _TO_DO_MANAGER_CVL2T2
TaskFlow is a modern and interactive task management web application designed to improve productivity. It allows users to create, manage, prioritize, and track tasks with features like deadlines, progress tracking, dark/light themes, drag-and-drop reordering, task notes, and data import/export — all stored locally in the browser.
✅ Add, edit, delete, and complete tasks
  
🎯 Priority & deadline management  

🌙 Dark / Light theme toggle

📊 Progress tracking with animations

📝 Task notes support

🔍 Search & filter tasks
  
📤 Import & export tasks (JSON)

🎉 Celebration effects on task completion
